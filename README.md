# Şehir Şantiyeciliği — Arayüz Mockup'ı

Şehir Şantiyeciliği bilgilendirme sitesinin statik arayüz mockup'ı. Üretim öncesi tasarım onayı için.

## Canlı Önizleme

https://by4r.github.io/sehirsantiyeciligi-com-template/

## Tech Stack

- Vanilla HTML / CSS / JS (jQuery yok, framework yok)
- FontAwesome 6.5 — line ikon seti
- Google Fonts — Saira Semi Condensed, Inter, JetBrains Mono
- Inline SVG sprite + blueprint hero

## Yapı

- `index.html` — One page (Header + 12 bölüm + Footer)
- `blog/sehir-santiyeciligi-nedir.html` — Blog detay iskeleti
- `assets/css/` — `tokens.css`, `base.css`, `layout.css`, `components.css`, `main.css`
- `assets/js/` — `nav.js`, `faq-accordion.js`, `standartlar-tabs.js`, `form.js`, `reveal.js`, `main.js`

## Not

Bu repo sadece arayüz incelemesi içindir. Üretim kodu (Faz 2 Laravel + admin panel) ayrı bir repo'dadır.
